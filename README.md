## Tech Stack
This project was engineered for speed and aesthetics, utilizing a lightweight, serverless architecture:

* **Core:** HTML5 (Semantic Structure)
* **Styling:** Tailwind CSS (via CDN for rapid styling & dark mode)
* **Animation Engine:** GSAP (GreenSock Animation Platform) + ScrollTrigger
* **Icons:** Remix Icon
* **Design System:** "Titanium & Deep Space" Aesthetic with Glassmorphism 2.0
